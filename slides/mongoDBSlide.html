Mongo DB - No SQL DB

Mongo Db is an open source software whihc can be downloaded from mongoDB.org. It is written in C++

Philosophy

Mongo DB Features

  • Standard DB Indexing
  • Documents are stored in BSON( binary JSON)
  • Any Valid JSON can be easily imported and queried
  • Schema-less, very flexible - no more blocking ALTER table
  • Makes for easy Horizontal Scaling
  • Map\Reduce, Very Fast, Super easy to install
  • Document Oriented - flexible
It is fast, scalable database that is suitable for many applications. It has tunable consistency, no tables and no schema to worry about and no impedence mismatch BSON is binary serizlization of JSON like objects.
This is extremely powerful which means mongo understands JSON natively

Benefits of using Mongo DB

  • Fast
  • Scalable
  • Tunable consistency
  • No tables
  • No Schema

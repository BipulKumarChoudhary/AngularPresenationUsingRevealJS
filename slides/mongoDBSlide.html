MongoDB - The leading No SQL DB

Mongo Db is an open source software which can be downloaded from mongoDB.org. It is written in C++

The Database Landscape - Philosophy

  • Mongo tries to achieve the performance of traditional key-value stores while maintaining functionality of traditional RDBMS

SqlDB comparison to MongoDB

Mongo DB Features

  • Standard DB Indexing
  • Documents are stored in BSON( binary JSON)
  • Any Valid JSON can be easily imported and queried
  • Schema-less, very flexible - no more blocking ALTER table
  • Makes for easy Horizontal Scaling
  • Map\Reduce, Very Fast, Super easy to install
  • Document Oriented - flexible
  • Efficient binary large objects via GridFS - stores images, videos, anything
It is fast, scalable database that is suitable for many applications. It has tunable consistency, no tables and no schema to worry about and no impedence mismatch BSON is binary serizlization of JSON like objects.
This is extremely powerful which means mongo understands JSON natively

Concepts - Cursors

  • Queries in MongoDB returns cursors instead of collections
  • A cursor allows us to iterate through the result set
  • A big reason for this is performance
  • Much more efficient then loading all objects in memory
  • The find() function will return the cursor obhject
  • 
                            var cursor = db.logged_requests.find({'status_code' : 200})
                            cursor.hasNext() // true
                            cursor.forEach(
                                function(item){
                                    print(tojson(item))
                                }
                            );
                            cursor.hasNext() // false
